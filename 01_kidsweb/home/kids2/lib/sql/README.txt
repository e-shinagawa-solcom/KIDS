	lib/sql 以下に配置する .sql ファイルは src/cmn/getmasterdata.php より呼び出さ
	れ、使用される事を前提としている。

	【ファイル命名規約】
	
	cnXXXXX 系の .sql は、コード＋名称の取得用に使用される。（テキスト情報取得）
	swXXXXX 系の .sql は、共通サブウィンドウの取得用に使用される。（<select>オブジェクト生成）



	【SQL出力カラム】

	○cn系
	cn系のsqlにおいて、出力カラムは下記の通り。

		1Col: ID
		2Col: name2

	出力カラムの割り当て値は、下記の通り。

		1Col: '[コード]'
		2Col: '[名称]'


	○sw系
	sw系のsqlにおいて、出力カラム順序が下記の通り決められている。
	これは、src/cmn/masterlib.js のJavaScriptファイル内においてもその内容を継承し
	ている。

		1Col: ID
		2Col: name2
		3Col: name3
		4Col: name4
		.............. 以降、必要に応じて自動拡張される

	出力カラムの割り当て値は、2Col、3Col、4Colを特殊な扱いとしていて、下記の通り。

		2Col: '[コード] [名称]'
		3Col: '[名称]'
		4Col: '[色]'
	
	4Colに指定する[色]は、3Colが存在しない場合のoption行のテキスト色としている。
	※これについては src/cmn/masterlib.js を参照のこと

*/
