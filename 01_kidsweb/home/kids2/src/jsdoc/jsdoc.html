{rval jsfiles} 
■Javascriptファイルの作成時の決まりごと
・各ファイルの先頭にファイルの概要を記述（下記の例１を参照）
・各関数の真上に関数の概要を記述（下記の例２を参照）
	1.概要、関数の引数の説明は、必ず記述
	2.ファイルの作成者と、関数の作成者が違う場合には、作成者の記述
	3.作成者以外の人が修正した場合には、更新者を記述
・各関数のHELP文の先頭は、「//@」を記述
　これを記述してないと関数への選択ができません
・「function」の必ず行の先頭に記述


例１ : 各ファイルの先頭に記述するファイルの概要
//: ----------------------------------------------------------------------------
//: ファイル概要: 共通関数
//: 備考        : 
//: 作成日      : YYYY/MM/MM
//: 作成者      : ** **
//: 修正履歴    : 
//: ----------------------------------------------------------------------------


例２ : 各関数の真上に記述する関数の概要
//@*****************************************************************************
// 概要   : 
// 引数   : 引数名, 説明
// 戻り値 : 
// 作成者 : 
// 更新者 : 
// 備考   : 
//******************************************************************************
{rval main}
